# ⚡ Focus Reader

A browser-based speed reading tool built for personal use — no frameworks, no dependencies, no installs. Just open the HTML file and read.

Built because I wanted a distraction-free way to get through books and documents faster. Ended up being a solid little project.

---

## Features

- **RSVP speed reading** — one word at a time, 60–800 WPM
- **Bionic focus letter** — one highlighted letter per word anchors your eye instantly
- **Fullscreen black mode** — zero distractions, press `F` to toggle
- **Book view panel** — visual page progress with a draggable bookmark; click any word to jump
- **File support** — loads `.txt`, `.pdf`, and `.docx` directly in the browser
- **No server, no upload** — everything runs locally, your files never leave your machine
- Keyboard shortcuts: `Space` play/pause · `← →` skip · `F` fullscreen

---

## Use it

Download `speed_reader_focus.html` → open in any browser. That's it.

```
No npm install. No build step. No account.
```

## Why I built this

I read a lot — books, papers, long docs. Most speed reading apps are paywalled or bloated. I wanted something clean, fast, and mine. Built entirely with vanilla HTML/CSS/JS in a single file.

Feel free to use it, fork it, or build on it.
